## 🌟 개요

Claude Code에서 즉시 실행할 수 있는 마케팅·영업 자동화 워크플로우 모음입니다. 단순 프롬프트가 아닌, 스크립트·스코어링 알고리즘·전문가 패널·자동화 파이프라인을 포함한 완전한 시스템입니다.

## 🛠 기술 스택

| 영역 | 기술 |
|------|------|
| **AI 런타임** | Claude Code (Anthropic) |
| **스크립트** | Python 3, Bash |
| **통합** | REST API, Webhook |

## 📦 스킬 카테고리

| 카테고리 | 설명 |
|---------|------|
| **Growth Engine** | 실험 자동화, 페이싱 알림, 주간 성과표 |
| **Sales Pipeline** | 방문자 → 파이프라인 전환, 딜 복원, 리드 생성 |
| **Content Ops** | 90점+ 콘텐츠 자동 생성, 품질 게이트 |
| **Outbound Engine** | ICP 정의 → 이메일까지 완전 자동화 |
| **SEO Ops** | 경쟁사 놓친 키워드 발굴, GSC 최적화 |
| **Finance Ops** | AI CFO 보고서, 비용 시나리오 모델링 |
| **Revenue Intelligence** | 콘텐츠 ROI 증명, 영업 콜 → 전략 전환 |
| **Conversion Ops** | 랜딩 페이지 CRO 감사, 설문 → 리드 마그넷 |
| **Podcast Ops** | 에피소드 1개 → 20+ 콘텐츠 자동 생성 |
| **Sales Playbook** | 가치 기반 가격책정, 콜 분석 |

## 🔍 사용법

```bash
git clone https://github.com/kimdzhekhon/ai-marketing-skills.git
cd ai-marketing-skills/<category>
pip install -r requirements.txt
cp .env.example .env
# .env에 API 키 설정 후 실행
```

Claude Code에서 `/skill <skill-name>` 으로 즉시 실행 가능합니다.
